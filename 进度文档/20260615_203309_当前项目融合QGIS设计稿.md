# 20260615_203309_当前项目融合QGIS设计稿

## 本次目标

不沿用 v2 设计稿，结合当前 WebQGIS 项目已经打开/实现的 UI 结构，以及本机 QGIS Desktop 的真实 UI 排版与操作逻辑，重新生成一份新版设计稿。

## 做了哪些

- 使用 Computer Use 成功识别并启动本机 `QGIS Desktop 3.44.10`。
- 观察到 QGIS 真实首屏结构：
  - 顶部菜单包含项目、编辑、视图、图层、设置、插件、矢量、栅格、数据库、Web、网孔、数据处理、帮助。
  - 顶部为多排密集工具栏，按钮会随图层和编辑状态启用/禁用。
  - 左侧有浏览器与图层停靠面板。
  - 浏览器树包含收藏夹、空间书签、主目录、本地磁盘、GeoPackage、SpatiaLite、PostgreSQL、Oracle、WMS/WMTS、XYZ 切片等入口。
  - 底部有坐标、比例尺、缩放、旋转角度、EPSG 等状态栏信息。
- 读取当前 WebQGIS 前端组件源码，确认当前项目真实 UI 能力：
  - `WebGisWorkbench.vue`：菜单栏、工具栏、上下文栏、左/中/右三栏、状态栏。
  - `DatasourcePanel.vue`：PostgreSQL 节点右键菜单、新建连接弹窗、保存并测试。
  - `LayerPanel.vue`：图层列表、图层右键菜单、图层样式编辑。
  - `MapCanvas.vue`：MVT 浏览、绘制/保存/删除/清空工具条。
  - `EditInspector.vue`：图层属性、属性表单、保存校验、SQL / 审计预览。
  - `AttributeTablePanel.vue`：属性表字段过滤、排序、字段元信息。
- 重新生成新版设计稿：
  - 新版 SVG 源稿：`设计稿/WebQGIS_当前项目融合QGIS设计稿.svg`
  - 新版 PNG 预览：`设计稿/WebQGIS_当前项目融合QGIS设计稿.png`
  - 新版 imagegen-ready prompt：`设计稿/WebQGIS_当前项目融合QGIS设计稿_imagegen_prompt.txt`
- 使用本地 Chrome headless 将新版 SVG 导出为 PNG。
- 使用图片预览验证新版设计稿渲染正常，未发现主要文本遮挡或空白问题。
