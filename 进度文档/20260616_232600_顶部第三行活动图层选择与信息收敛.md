# 顶部第三行活动图层选择与信息收敛进度文档

## 需求

1. 顶部第三行中的活动图层改为 `select`，可选择当前活动图层。
2. 顶部第三行移除“编辑”显示。
3. 顶部第三行“捕捉”只在开启时显示。
4. 顶部第三行移除“显示链路”和“编辑链路”信息。

## 本次实现

- 将工作台第三行“活动图层”从只读文本改为下拉选择框：
  - 选项来自已加载图层 `layers`。
  - 当前值绑定 `activeLayerId`。
  - 切换时调用既有 `workspace.setActiveLayer`，继续复用现有活动图层与编辑状态联动逻辑。
- 移除第三行编辑状态 badge：
  - 删除 `activeLayerLabel` 和 `activeLayerEditStatus` 两个不再使用的派生状态。
  - 第三行不再显示“编辑: 关闭/编辑中/只读”。
- 调整第三行捕捉信息：
  - 仅当 `isSnapEnabled` 为 true 时显示“捕捉: 顶点 + 线段, 8 px”。
- 移除第三行链路说明：
  - 不再显示“显示链路: MVT”。
  - 不再显示“编辑链路: 原始 PostGIS geometry”。
